# Verilog_Codes

This repository contains Verilog HDL programs developed for learning and practicing Digital Design concepts.  
It includes implementations of combinational and sequential circuits along with simulation testbenches.  
The designs are written using behavioral, dataflow, and structural modeling techniques.  
Simulation and functional verification are performed using QuestaSim 10.7c.  
The repository is regularly updated with new Verilog programs and digital design examples.  
It serves as a reference for students, VLSI beginners, and design verification learners.
